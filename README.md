## 📌 What this is

google-dorking is a search helper for finding content that normal web searches can miss. It brings together Google dorks and related search methods for Bing, Yandex, Shodan, Tor, reverse image search, and metadata checks.

Use it to look for public files, exposed pages, hidden paths, documents, and other open web data. It is built for end users who want a simple way to follow proven search patterns without building them by hand.

## 🖥️ What you need

- A Windows computer
- An internet connection
- A web browser
- Enough free disk space for the app and any saved data
- Permission to search the sites and data you plan to review

The app is meant to be light and easy to run on a normal Windows setup.

## 🧰 Basic search workflow

A simple workflow looks like this:

1. Think of what you want to find
2. Choose the right search engine or source
3. Use a query pattern from the app
4. Search for file types, names, or page paths
5. Open results that match your goal
6. Save useful queries for later

Example query ideas:

- site:example.com filetype:pdf
- intitle:index of
- inurl:admin
- filetype:xls confidential
- ext:log
- "password" "login"

Use these patterns with care and only on public data you are allowed to review
